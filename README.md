# cellai
Windows-Desktop-Anwendung zur automatisierten Analyse von Mikroskopaufnahmen.

## Features
- Automatische Zellerkennung mit TensorFlow
- GUI mit Live-Kamera-View
- Export als .exe

## Installation
1. Python 3.8+ installieren
2. `pip install -r requirements.txt`
3. `python main.py`
